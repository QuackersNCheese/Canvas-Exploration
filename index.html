Drawing with Canvas

I'm using an MDN Web Docs tutorial as a guide to build a simple website to learn how to draw with the Canvas API. I've made an .html, a .css, and a .js file to create and control a canvas element. I plan to make this document to explain how I'm building it, as I am inspired to build it by the tutorial. (with my own flair, of course) It seems like github copilot wants to help too, so I'm letting it.

Setting Up

The HTML document doesn't need much more than a canvas element and links to CSS and JavaScript. I adjust the CSS file as I write in the HTML file when I want to make it look good. The JavaScript file is where all of the programmatic drawing occurs. At first, most of the drawing is in a draw function. Running the draw function happens through the event listener at the bottom of the .js file.

I'm using the getContext() method to get a 2d rendering context for the drawing surface of the canvas, and then drawing shapes with various properties as examples.

Rectangles

I'm using the fillStyle property to set the color of the fill, and the fillRect() method to draw a filled rectangle on the canvas. I'm using the strokeStyle property to set the color of the stroke, and the strokeRect() method to draw a rectangle outline on the canvas.

I'm using the clearRect() method to clear the canvas. I'm using the beginPath() method to begin a path, the moveTo() method to move the starting point of a new sub-path to the (x, y) coordinates, the lineTo() method to add a straight line to the current sub-path, the stroke() method to draw the path, and the fill() method to fill the path. I'm using the arc() method to create an arc, the fillText() method to draw filled text on the canvas, and the strokeText() method to draw outlined text on the canvas. I'm using the drawImage() method to draw an image on the canvas. I'm using the createLinearGradient() method to create a linear gradient, the createRadialGradient() method to create a radial gradient, and the createPattern() method to create a pattern.
